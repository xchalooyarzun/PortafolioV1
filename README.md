# 🧠 CoreReset-GAP — Portafolio Profesional

Portafolio web desarrollado con enfoque en desarrollo moderno y aplicaciones orientadas a salud.

Proyecto personal que muestra habilidades en construcción de interfaces, validación de datos y experiencia de usuario tipo software real.

---

## 🏥 Proyecto Destacado

Sistema clínico web enfocado en:

- Seguridad del paciente
- Validación de datos
- Registro de signos vitales
- Interfaz clara y funcional

---

## 💻 Tecnologías utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

---

## 🚀 Cómo ejecutar el proyecto

```bash
npm install
npm run dev
```

Abrir en navegador:

http://localhost:5173

🎯 Objetivo

Combinar tecnología y área de salud para construir software útil, seguro y profesional.
